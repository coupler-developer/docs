## 롤백 기준

- Contracts package `0.1.2`는 이미 발행된 package artifact이므로 삭제/재발행으로 되돌리지 않는다.
- Consumer bump PR merge 전 문제가 확인되면 Admin/Mobile branch의 dependency pin을 `0.1.1`로 되돌리고 lockfile을 다시 생성한다.
- Consumer bump PR merge 후 문제가 확인되면 각 소비자 repo에서 revert PR을 만들어 `@coupler-developer/coupler-api-contracts@0.1.1` lockfile 상태로 복구한다.
- 운영 API/Admin 배포와 Mobile Store/NextPush는 이번 릴리스 범위가 아니므로 rollback 대상이 아니다.
- API contract runtime cutover가 필요한 변경이 추가되면 별도 릴리스 기록에서 `apiContractCutover` evidence를 채우고 N+1/legacy traffic block 기준을 다시 검증한다.
