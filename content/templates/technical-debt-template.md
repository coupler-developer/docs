# Technical Debt 템플릿

## 문서 역할

- 역할: `부채`
- 문서 종류: `technical-debt`
- 충돌 시 우선 문서: 해당 항목이 참조하는 규범 문서
- 기준 성격: `as-is`

## 목적

- 현재 확인된 기술 부채를 우선순위와 근거 중심으로 기록한다.

## 적용 범위

- 평가 대상 레포/서비스 범위를 명시한다.
