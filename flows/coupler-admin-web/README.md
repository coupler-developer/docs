# Coupler Admin Web Flows

이 디렉토리는 Coupler Admin Web 관련 플로우 문서를 저장합니다.

## 문서 작성 가이드

각 플로우 문서는 다음 형식을 따릅니다:

1. **제목**: 관리자 기능의 주요 목적
2. **개요**: 플로우에 대한 간단한 설명
3. **UI 구성**: 화면 구성 및 컴포넌트
4. **사용자 액션**: 관리자가 수행하는 작업
5. **API 연동**: 사용되는 API 엔드포인트
6. **권한**: 필요한 관리자 권한
7. **주의사항**: 개발 및 운영 시 주의사항

## 플로우 목록

- 사용자 관리 플로우
- 시스템 설정 플로우
- 데이터 모니터링 플로우
