# Coupler API Flows

이 디렉토리는 Coupler API 관련 플로우 문서를 저장합니다.

## 문서 작성 가이드

각 플로우 문서는 다음 형식을 따릅니다:

1. **제목**: API 플로우의 주요 기능
2. **개요**: 플로우에 대한 간단한 설명
3. **엔드포인트**: 관련 API 엔드포인트 목록
4. **요청/응답**: 데이터 구조 및 예시
5. **비즈니스 로직**: 주요 처리 로직
6. **데이터베이스**: 사용되는 테이블 및 쿼리
7. **에러 처리**: 예외 상황 및 에러 코드

## 플로우 목록

- 인증/인가 플로우
- 데이터 CRUD 플로우
- 배치 처리 플로우
