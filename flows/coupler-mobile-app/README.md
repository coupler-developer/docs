# Coupler Mobile App Flows

이 디렉토리는 Coupler Mobile App 관련 플로우 문서를 저장합니다.

## 문서 작성 가이드

각 플로우 문서는 다음 형식을 따릅니다:

1. **제목**: 플로우의 주요 기능이나 목적
2. **개요**: 플로우에 대한 간단한 설명
3. **단계**: 플로우의 각 단계별 상세 설명
4. **관련 컴포넌트**: 연관된 UI 컴포넌트나 모듈
5. **API 연동**: 사용되는 API 엔드포인트
6. **주의사항**: 개발 시 주의해야 할 사항

## 플로우 목록

- 사용자 인증 플로우
- 데이터 동기화 플로우
- 푸시 알림 플로우
