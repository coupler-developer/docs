# Cross-Project Flows

이 디렉토리는 여러 프로젝트(Mobile App, API, Admin Web)를 아우르는 통합 플로우 문서를 저장합니다.

## 문서 작성 가이드

통합 플로우 문서는 다음 형식을 따릅니다:

1. **제목**: 통합 플로우의 주요 기능
2. **개요**: 전체 플로우에 대한 설명
3. **참여 시스템**: 관련된 모든 시스템 (Mobile App, API, Admin Web)
4. **플로우 다이어그램**: 시스템 간 상호작용 흐름
5. **단계별 설명**:
   - Mobile App에서의 동작
   - API에서의 처리
   - Admin Web에서의 관리
6. **데이터 흐름**: 시스템 간 데이터 전달 방식
7. **에러 처리**: 각 시스템별 에러 처리 방법

## 플로우 목록

- 사용자 등록 및 인증 통합 플로우
- 데이터 동기화 통합 플로우
- 알림 발송 통합 플로우
